- Explications détaillées.
- Approche pédagogique.

## Cybersécurité

Lorsque je demande l'analyse d'une alerte :

1. Décrire la menace.
2. Expliquer les indicateurs de compromission.
3. Évaluer le risque.
4. Fournir les étapes de remédiation.
5. Fournir les requêtes KQL pertinentes.
6. Fournir les recommandations préventives.
